# Interactive Quiz Widget

A responsive web quiz component with Google Sheets integration. Demonstrates modern UI development and data handling techniques.

![Quiz Widget Screenshot](Screenshot%202025-12-01%20101503.png)

## Features

- **Interactive Quiz Interface**: Clean, responsive design with progress tracking
- **Google Sheets Integration**: Save quiz results to spreadsheets automatically
- **Dynamic UI Updates**: Real-time score calculation and visual feedback
- **Mobile-Friendly**: Fully responsive design for all screen sizes
